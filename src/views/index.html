Overview

Detta UI toolkit kan du använda dig av för att snabbare kunna skapa upp sidor med färdiga komponenter som t.ex knappar, inputfält, formulär.

I denna style guide finns kom-igång information samt exempelkod på alla komponenter som finns tillgängliga.

Strukturen följer metodiken atomic design.

Vad är atomic design

Atomic design är en metodik för att skapa ett design system.

Det finns fem nivåer och dessa är:

  • Atoms
  • Molecules
  • Organisms
  • Templates
  • Pages

Läs mer om atomic design här.
